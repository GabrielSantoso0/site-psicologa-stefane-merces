02 — ASSINATURA VISUAL

O Logotipo

O símbolo é um anel octogonal segmentado — 8 formas de precisão formando um sistema unificado. Ciano nos vértices, branco nos planos principais. Representa a Sincronia dos pontos de contato de um negócio.

↑ Versão principal — fundo escuro

↑ Versão alternativa — fundo claro

Regras de Uso

✔ Use sempre com espaço de proteção equivalente à altura do símbolo em todos os lados.

USO CORRETO

✔ Em fundos coloridos, use a versão monocromática branca ou preta conforme contraste.

USO CORRETO

✗ Nunca distorça ou altere as proporções do símbolo.

PROIBIDO

✗ Não aplique sobre fundos com pouco contraste ou em fotografia sem overlay.

PROIBIDO

Tamanho Mínimo

Símbolo isolado: 24px digital / 8mm impresso. Assinatura horizontal completa: 120px digital / 35mm impresso.

04 — SOCIAL MEDIA

Presença Digital

Visual Generator (Design Squad) — Os posts da Sincronia seguem 3 grandes arquétipos de conteúdo, cada um com dimensões e intenção específica. Nenhum post usa cores vibrantes concorrentes. O fundo é sempre escuro.

Post de Autoridade · 1:1
Frases de posicionamento e liderança

Post de Processo · 4:5
Metodologia e resultado do cliente

Story / Capa · 9:16
Lançamentos e propostas premium

Regras de Conteúdo Visual

  • Sempre escuro: Fundo preto ou muito escuro (#0A–#1A). Nenhuma exceção.
  • Uma cor de acento: Use apenas Cyan OR Blue por post. Nunca os dois com mesmo peso.
  • Tipografia limpa: Máx. 2 estilos por peça. Body em Plus Jakarta Sans, dados em Geist Mono.
  • Logo sempre presente: Posição inferior esquerda ou superior direita. Nunca centralizado por cima do texto.
  • Sem stock photos: Use ilustrações abstratas, capturas de UI real ou renders 3D alinhados ao universo tech.

05 — UI / PRODUTO DIGITAL

Previsão de Interface

UX Designer × Visual Generator (Design Squad) — A Sincronia terá uma plataforma de gestão de automações para seus clientes. As interfaces devem expressar a mesma sofisticação do Brand Book.

Dashboard Web · Painel de Automações Ativas

App Mobile · Status do Sistema

Princípios de Interface (UX Designer)

  • Dark mode exclusivo: A UI da Sincronia não tem modo claro. O dark mode não é estética — é posicionamento.
  • Dados como protagonistas: Métricas, porcentagens e status devem ser a primeira coisa que o olho encontra.
  • Glassmorphism moderado: Painéis com fundo semi-transparente + borda sutil. Não exagerar.
  • Micro-animações funcionais: Apenas onde reforçam o estado do sistema. Transitions 0.4s ease-out.
  • Acessibilidade mínima WCAG AA: Contraste de texto ≥ 4.5:1. Focus ring visível em todos os inputs.
